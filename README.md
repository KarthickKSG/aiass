## üöÄ Deployment

King AI is architected for high-performance edge deployment.

1. **Initialize API Key**: Obtain a Gemini API key from [Google AI Studio](https://aistudio.google.com/).
2. **Environmental Variables**: Assign the key to `process.env.API_KEY`.
3. **Run Production Node**:
   ```bash
   npm install
   npm run build
   npm run preview
   ```
